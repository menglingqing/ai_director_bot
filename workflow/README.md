# 工作流说明

本目录定义 ai_director_bot 的流水线：各阶段的顺序、每阶段调用的技能、以及输入输出的约定。

**与 OpenClaw 的协作方式**（谁出提示词、谁生成、谁审核、何时进入下一步）见：[工作流设计-与OpenClaw协作方式.md](../docs/工作流设计-与OpenClaw协作方式.md)。

---

## 默认流水线：`default.yaml`

- **用途**：供 ai_director_bot 或上层调度逻辑读取，按阶段依次调用技能。
- **结构**：阶段（stage）→ 步骤（steps）→ 技能 ID、输入来源、输出产物。

使用方式（规划）：

- 手动：按 YAML 中的阶段顺序，在对话中依次请求对应技能。
- 自动：由 ai_director_bot 解析 `default.yaml`，根据当前已有的产出物（剧本、分镜表、分镜图等）决定下一步调用哪个技能，或跑完整条流水线。

---

## 阶段与技能映射

| 阶段 | 技能 ID | 技能名称（简要） | 主要输入 | 主要输出 |
|------|---------|------------------|----------|----------|
| S0 | s0-script | 剧本创作 | 用户创意 | 剧本 |
| S1 | s1-shotlist | 分镜表 | 剧本 | 分镜表（结构化） |
| S2 | s2-storyboard, s2-character, s2-scene | 分镜概念、角色卡、场景设定 | 分镜表、剧本 | 分镜概念图、角色卡、场景提示词 |
| S3 | s3-costume, s3-refine | 服化道、垫图完善分镜 | 分镜概念图、角色/场景 | 服化道图、定稿分镜图 |
| S4 | s4-camera, s4-beat | 运镜、音乐卡点 | 分镜表、定稿分镜、BGM | 运镜提示词、卡点提示词 |
| S5 | s5-video, s5-patch, s5-style | 视频生成、单镜修改、风格统一 | 定稿分镜、运镜、修改意图、参考图 | 视频提示词、迭代提示词、画风描述 |

---

## 迭代与分支

- **单镜/局部修改**：任意阶段若对某一张分镜图不满意，可单独调用 `s5-patch`，输入「当前图 + 修改意图」，输出迭代用提示词，再生成新图。
- **风格统一**：若希望全片统一画风，可在 S2 之前或之后调用 `s5-style`，将输出的画风描述注入到后续所有分镜/服化道/场景提示词中。
- **只做到分镜**：流水线可在 S3 结束后停止，不调用 S4/S5，仅产出定稿分镜图。
