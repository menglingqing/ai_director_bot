# ai_director_bot（AI 导演助手）

从创意到分镜再到成片，一站式调配剧本、分镜、服化道、运镜与视频生成等技能，陪你完成视频制作全流程。**所有工作都在与 OpenClaw 的对话中完成**：助手出提示词或直接出图/出片，你选择「自己拿提示词去别的站生成」或「助手生成、你审核」，审核通过后再进入下一步。

**Telegram Bot**：[@mlqDirectorbot](https://t.me/mlqDirectorbot)（导演助手对话入口）

详见：[工作流设计 - 与 OpenClaw 协作方式](docs/工作流设计-与OpenClaw协作方式.md)。

---

## 项目目标

- **工作流**：定义并实现「创意 → 剧本 → 分镜表 → 分镜概念图 → 服化道 → 定稿分镜 → 运镜/卡点 → 视频」的标准化流水线。
- **技能**：为每个环节提供可被调度的技能（提示词生成或与现有技能联动），便于按阶段调用与迭代。
- **可扩展**：技能与工作流解耦，便于后续新增阶段或替换实现。

---

## 快速开始

### 环境与依赖

- 需要能运行本 Bot 或对接 Agent 的环境（如 OpenClaw、Cursor 等）。
- 技能可能依赖 Cursor/Claude 或本地技能目录（如 `.cursor/skills`），见 [技能清单](docs/技能清单.md)。

### 与 OpenClaw 的两种协作方式

1. **仅要提示词**：助手生成提示词，你复制到即梦、Midjourney、Seedance 等平台自己生成图/视频，需要时把结果带回或说「继续」。
2. **助手生成 + 你审核**：助手直接产出图/视频，你审核后说「通过」再进入下一步，或说明修改意见让助手重做。

每个出图/出片的步骤都设**审核闸门**，只有你说通过后助手才进入下一阶段。详见 [工作流设计](docs/工作流设计-与OpenClaw协作方式.md)。

### 使用方式（规划）

- **按阶段推进**：根据 [工作流](workflow/README.md) 顺序，在对话中依次触发对应技能（剧本 → 分镜表 → 分镜概念 → 服化道 → 定稿分镜 → 运镜/卡点 → 视频）。
- **一键流水线（待实现）**：根据当前阶段与产出物自动调度下一技能，或跑完整条流水线。

## 工作流概览

```
用户创意
    ↓
[S0] 剧本创作 → 剧本
    ↓
[S1] 分镜表/镜头列表
    ↓
[S2] 角色卡 + 场景设定 + 分镜概念图（按镜）
    ↓
[S3] 服化道提示词 → 服化道参考图 → 垫图完善分镜图 → 定稿分镜
    ↓
[S4] 运镜提示词（按镜）+ 音乐卡点（可选）
    ↓
[S5] 视频生成提示词 → 成片（可选）；单镜修改 / 风格统一（迭代）
```

详细阶段与技能见 [docs/技能清单.md](docs/技能清单.md)。

## 与现有技能的关系

本仓库中的技能实现或提示词模板，会与你在 AIGC 项目下已有的 Cursor/Claude 技能配合使用，例如：

| 阶段/用途     | 可复用资源 |
|--------------|------------|
| 分镜概念     | cinematic-prompt, magazine-interior-photo |
| 服化道/人像  | oiran-photoshoot-prompt, nano-banana-interior |
| 运镜         | seedance-camera-prompt |
| 音乐卡点     | music-beat-sync-prompt |

ai_director_bot 负责**按阶段调度**这些能力，并补全「分镜表、角色卡、场景设定、视频生成提示词、单镜修改」等新技能。
