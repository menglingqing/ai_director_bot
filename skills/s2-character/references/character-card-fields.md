# 角色卡 · 高信息量字段速查

角色卡/角色参考图提示词需覆盖以下维度，确保多镜头下**视觉一致**且**可直接用于文生图/图生图**。每维度用**具体、可复现**的表述，避免「好看」「有气质」等抽象词。

| 维度 | 必含要点 | 示例表述 |
|------|----------|----------|
| **身份与年龄** | 年龄段、地域/族裔、职业或身份标签 | 22–25 岁韩国女偶像；30 岁左右北欧男工程师 |
| **场景/环境** | 具体地点、背景元素、虚化程度 | 质朴木屋门口，背景虚化的远处连绵山脉 |
| **服装·上身** | 颜色、图案、领型、袖型、材质 | 蓝色碎花迷你裙，白色小花图案，V 领露出锁骨 |
| **服装·下身与鞋** | 裙/裤长度、鞋型与颜色、与身材关系 | 过膝驼色麂皮靴，衬托修长双腿 |
| **配饰** | 耳环、项链、手饰等 + 与光线的互动 | 金色耳环在晨光中闪动，细银链贴在白皙皮肤上 |
| **五官·眼** | 眼型、眼尾走向、眼神方向、眼神光位置 | 杏仁眼、眼尾微扬；不直视镜头，朝镜头外约 15 度望远山；虹膜 11 点钟方向眼神光 |
| **表情与叙事** | 情绪区间、唇部状态、未说出的叙事 | 介于渴望与平静接受之间，双唇微张，笑意将起未起 |
| **姿态与身体** | 相对镜头角度、重心、轮廓线、手/脚位置 | 四分之三侧对镜头，一手轻触门框，重心一侧，轻松 S 形轮廓；头略偏，下巴微抬 |
| **主光** | 方向、角度、光质（硬/软/漫射）、来源 | 柔和晨光约 45 度从木窗照入，经薄云漫射 |
| **补光与光比** | 补光来源、光比、阴影通透度 | 原木墙面补光，光比约 2:1，阴影通透、肤色透亮 |
| **轮廓光** | 方向、作用部位、与背景分离 | 门框方向轮廓光，发丝和肩部金色边缘，与背景分离 |
| **镜头与景深** | 等效焦距、光圈、景深效果、镜头风格 | 等效 85mm、f/2.0，景深极浅，近侧眼清晰、远侧耳与发梢柔焦；Planar 风格光学柔焦 |
| **颗粒与质感** | 胶片/颗粒型号与强度、分布 | 柯达 Portra 160 颗粒约 0.22，均匀分布，中灰区可见 |
| **色彩基调** | 冷暖对比、高光/阴影色偏、饱和度 | 老木暖蜜色与远山冷蓝灰对比，蓝裙衔接；高光略暖奶油，阴影通透带冷蓝；饱和度中低 |
| **构图与取景** | 人物在画中位置、视线引导、取景范围 | 人物位于画面右三分之一，视线引向左侧留白；从大腿中部向上取景；轻微暗角 |
| **整体风格** | 3–5 个锚定词 | 电影感、编辑感、有灵魂、性感而克制 |

## 身份一致性（图生图/多镜头）

若后续用**角色参考图**垫图生成分镜，在角色卡中注明：

- **严格保留**：脸型、五官比例、年龄感、性别、肤色、标志性特征（痣、疤痕等）
- **允许变化**：妆发、服装、姿势、环境、灯光、景别
- **禁止**：换脸、网红美颜、动漫风格化、过度磨皮、多指/畸形

## 与剧本/分镜的衔接

- 从剧本或分镜表提取：角色姓名、职业、关键戏份、情绪弧线、出现的场景与服装变化。
- 同一角色多套服装时，可为每套生成一条「服装变体」描述，共用同一「身份与五官」基础描述。
