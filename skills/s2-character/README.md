# S2：角色设定/角色卡提示词生成

**阶段**：分镜概念与视觉设定  
**状态**：✅ 已实现（见项目 skill）

## 输入

- 剧本或分镜表（人物设定摘要）

## 输出

- **高信息量**角色卡：身份年龄、场景、服装配饰、五官表情、姿态、光影、镜头质感、色彩构图等维度
- 「角色参考图」生成提示词（一段话，可直接用于文生图/图生图），保证多镜头下角色视觉一致

## 在流水线中的位置

- 与 s2-scene 并行或略早，产出 `character_cards`，供 S2 分镜概念与 S3 垫图完善使用。

## 实现方式

- **技能内容在本目录**：`SKILL.md`（使用说明与质量标杆）+ `references/character-card-fields.md`（维度与字段速查）。
- 按「身份与年龄 → 场景 → 服装 → 配饰 → 五官与眼神 → 表情与叙事 → 姿态 → 光影 → 镜头与质感 → 色彩与构图 → 整体风格」等维度生成**可复现、摄影级**的角色描述，避免角色卡信息过少导致多镜不一致。
- OpenClaw 导演助手在完成剧本与分镜表后，创建角色卡时应**调用或遵循**本目录 SKILL.md 与 references 的维度与输出格式。

## 角色卡模板约定

- 结构化角色卡：各维度分块或表格，便于按「角色 ID + 维度」引用。
- 可选：为同一角色多套服装维护多条「服装变体」描述，共用同一「身份与五官」基础。
- 若后续垫图：在角色卡中注明「严格保留 / 允许变化 / 禁止」项，供 S3-2 垫图完善使用。
