# 角色卡 / 角色参考图提示词生成

为剧本或分镜表中的角色生成**高信息量**角色卡与可直接用于文生图/图生图的「角色参考图」提示词。目标：**描述足够具体**，使同一角色在多镜头、多场景下视觉一致，且单段提示词即可驱动高质量人像生成。

## 何时使用

- OpenClaw 导演助手执行 **S2-2 角色设定/角色卡** 时
- 用户提到「角色卡」「角色设计」「角色参考图」「角色一致性」
- 用户反馈「角色描述信息太少」「生成的角色不像同一个人」

## 输入与输出

| 项目 | 说明 |
|------|------|
| **输入** | 剧本或分镜表（含人物设定摘要）：角色姓名、职业/身份、关键戏份、出现场景、服装与情绪 |
| **输出** | ① 结构化角色卡（便于多镜引用）；② 一段可直接粘贴的「角色参考图」生成提示词（中英可选） |

## 核心原则

1. **具体优于抽象**：用「杏仁眼、眼尾微扬、11 点钟方向眼神光」而非「眼睛好看」；用「过膝驼色麂皮靴」而非「靴子」。
2. **可复现**：五官、服装、光影、镜头参数写清，便于后续分镜与垫图时复用。
3. **叙事与气质**：用「介于渴望与平静接受之间、双唇微张、笑意将起未起」等表述，把情绪与未说出的叙事写进描述。
4. **摄影级控制**：包含主光方向、光比、轮廓光、等效焦距与光圈、颗粒/质感，使角色参考图具备电影感/编辑感。

## 使用步骤

1. **从剧本/分镜表提取角色信息**
   - 角色姓名、年龄区间、身份/职业、与剧情相关的气质或弧线
   - 出现的场景与典型环境（室内/室外、时代感、氛围）
   - 剧本中提到的服装或造型（若有多场戏，可列出 1～2 套代表造型）

2. **按维度填写角色卡与提示词**
   - 必选维度见本目录 `references/character-card-fields.md`，逐项用**具体、可复现**的表述填写。
   - 维度顺序建议：身份与年龄 → 场景/环境 → 服装（上身→下身与鞋）→ 配饰 → 五官与眼神 → 表情与叙事 → 姿态与身体 → 主光 → 补光与光比 → 轮廓光 → 镜头与景深 → 颗粒与质感 → 色彩基调 → 构图与取景 → 整体风格。
   - 若剧本未给出某维度，按角色身份与戏份**合理推断**并写明，避免留空。

3. **输出两种形式**
   - **角色卡（结构化）**：以表格或分块形式列出各维度，便于后续 S2-1 分镜概念、S3 服化道、S3 垫图完善时按「角色 ID + 维度」引用。
   - **角色参考图提示词（一段话）**：将上述维度串联成一段流畅、可直接用于文生图的提示词；若目标平台为英文模型，可同时给出英文版。

4. **身份一致性说明（若后续会垫图）**
   - 在角色卡末尾注明：严格保留（脸型、五官、年龄感、肤色等）；允许变化（妆发、服装、姿势、环境）；禁止（换脸、网红美颜、动漫化、过度磨皮、畸形）。便于 S3-2 垫图完善时约束一致性。

## 质量自检

- [ ] 身份与年龄、场景、服装、配饰都有**具体**描述（颜色、材质、款式）？
- [ ] 五官与眼神是否包含：眼型、视线方向、眼神光位置、表情与叙事感？
- [ ] 姿态是否包含：相对镜头角度、重心、轮廓线（如 S 形）、手/脚位置？
- [ ] 光影是否包含：主光方向与光质、补光与光比、轮廓光？
- [ ] 是否包含镜头/景深/颗粒/色彩/构图等摄影级参数？
- [ ] 整段提示词读下来能否在脑中形成**清晰、可复现**的画面？

## 参考示例（质量标杆）

以下为「高信息量」角色参考图提示词的标杆风格，生成时请达到相近密度与具体度：

> 一位 22–25 岁的韩国女偶像，站在一座质朴木屋门口，背景虚化的远处是连绵山脉。她身穿蓝色碎花迷你裙，白色小花图案，V 领露出锁骨，搭配过膝驼色麂皮靴，衬托修长双腿。金色耳环在晨光中闪动，细银链贴在白皙皮肤上。杏仁眼、眼尾微扬，眼神深邃而若有所思——不直视镜头，而是朝镜头外约 15 度望向远山，像在出神或回忆。虹膜在 11 点钟方向有一处眼神光。表情里有一种未说出的叙事——介于渴望与平静接受之间，双唇微张，笑意将起未起。身体呈约四分之三侧对镜头，一手轻触斑驳木门框，重心自然落在一侧，形成轻松的 S 形轮廓；头略偏，下巴微抬。柔和晨光以约 45 度从木窗照入，经薄云漫射。原木墙面提供温暖补光，光比约 2:1，阴影通透、肤色透亮。门框方向有一道轻微轮廓光，在她发丝和肩部勾勒出金色边缘。等效 85mm、f/2.0，景深极浅，复古人像镜头光学柔焦。柯达 Portra 160 胶片颗粒，饱和度中等偏低。人物位于画面右三分之一，视线将观者引向左侧留白。整体风格：电影感、编辑感、有灵魂、性感而克制。
