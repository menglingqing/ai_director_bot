# 技能目录

本目录按制片阶段组织 ai_director_bot 所需技能。每个子目录对应一个或一组技能，内含：

- **SKILL.md**：技能说明、触发条件、输入输出、与 Bot 的对接方式（待实现时补充）。
- **references/**：提示词模板、示例、可选参考文档。
- 若技能复用时，可只保留 README 说明并指向外部技能路径（如 `seedance-camera-prompt`）。

---

| 目录 | 阶段 | 技能 | 状态 |
|------|------|------|------|

| s2-character | S2 | 角色设定/角色卡提示词生成 | ✅ 已有 |
| s2-scene | S2 | 场景/环境设定提示词生成 | 🆕 待建 |
| s3-costume | S3 | 依靠分镜概念图定服化道的提示词生成 | ✅ 已有 |
| s3-refine | S3 | 依靠服化道等信息垫图完善分镜图的提示词生成 | ✅ 已有 |
| s4-camera | S4 | 分镜 ↔ 运镜提示词（联动） | 🔗 联动 |
